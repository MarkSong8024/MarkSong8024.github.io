# fs模块

# fs常用api

node的核心模块

  • .access
    判断文件是否存在,不存在则就报错

  • .exists
    判断文件是否存在,不存在则返回false[已废弃]

  • .readFile
    读取文件.如果文件特别大,可能会导致内存溢出,64k以下的文件都可以使用该方法

WARNING

以下为文件流操作, 所需步骤

  • .open
    文件操作流,步骤一.先打开文件
let fs = require('fs')
let path = require('path')
fs.open(path.resove(__dirname, './name.txt), (err, fd) => {
    ...
    // fd 文件描述符. mac从20开始. window从3开始
})

  • .read
    文件操作打开后,读取文件

  • .write
    文件操作打开后,写文件

  • .close
    文件操作结束,关闭文件
